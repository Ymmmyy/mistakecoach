# Data Notes

This GitHub package includes only small demo data and lightweight processed summaries.

The original Eedi raw data is not included because it is large. To reproduce the full experiment, place the Eedi files outside this project using:

```text
../data/
  train_data/train_task_1_2.csv
  test_data/test_public_answers_task_1.csv
  metadata/question_metadata_task_1_2.csv
  metadata/student_metadata_task_1_2.csv
  metadata/subject_metadata.csv
```

The final report uses only the public test file:

`test_data/test_public_answers_task_1.csv`
